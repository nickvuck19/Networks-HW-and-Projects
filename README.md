# Networks-HW-and-Projects
Homework and Projects from Intro to Networks and their Applications.
- HW1.py is a simulator that emulates the behavior of hosts and routers by maintaining some state associated with each router, and this state is updated based on
the actions that hosts, links, and routers take.
- The files chat.py, client2,py, client_test.py, dirservice.py, and server_test.py simulates a messaging system by implementing a chat client that is used to send
messages, and a directory server that maps a user identifer to an IP address. The chat clients talk with the directory server using TCP, and they talk with each other by using UDP.
- The wireless_simulator.py file utilizes random media access protocols for wireless communication. Both Slotted ALOHA and CSMA/CA are used, and the simulator displays the behaviors of these protocols when there are multiple nodes that share a one-hop channel under different trafic conditions
